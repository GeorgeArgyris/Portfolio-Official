Digital Twin Model based on Neural Complex Event Recognition

This is my diploma thesis project presenting a full-stack real-time monitoring framework built around a 3D digital twin of an electrical infrastructure. Sensor data streams from multiple devices via WebSockets and Apache Kafka into an Apache Flink pipeline, where a rule-based engine first detects simple events, then feeds them into a trained LSTM neural network — powered by TimeGAN-generated synthetic sequences — to recognize complex, multi-layer fault patterns. Higher-level events are escalated and resolved entirely within Flink's DataStream API, keeping latency minimal even at scale. The entire processed state is rendered live inside a Unity 3D digital twin, giving operators an interactive spatial view of their installation with real-time alerts and drill-down analysis. Throughput scales proportionally with Flink parallelism, and a user study confirmed the interface meaningfully supports decision-making in large-scale electrical environments.
